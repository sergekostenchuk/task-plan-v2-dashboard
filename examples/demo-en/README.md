# TASK-PLAN Dashboard Demo

This workspace is the English demo fixture for the Antigravity `Task Plan Dashboard` extension.

Open this folder in Antigravity and use the `Task Plan` activity bar item.

Files:

- `FEATURE-PREPARATION.md` - readiness gate
- `TASK-PLAN.md` - canonical plan
- `.task-plan/events.jsonl` - agent event feed
- `artifacts/` - example code, review, and test artifacts

Use this workspace when you want:

- an English-language walkthrough of the dashboard
- an example `TASK-PLAN v2` with 10 tasks
- a realistic event timeline and artifact links
